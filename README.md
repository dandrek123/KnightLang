# KnightLang

KnightLang is a programming language built from scratch in Python.

## Current Features

- Variables
- Integers
- Strings
- Arithmetic (+, -, *, /)
- String concatenation
- Conditional evaluation
- Lexer

## Example

let name = "DAndre"

print("Hello " + name)

## Roadmap

- Better lexer
- Parser
- Real if blocks
- Loops
- Functions
- Compiler
